Media queries

Media queries create responsive designs that adapt to different devicdes and screen sizes, providing a better user experience acros the range of platfrom and devices.

  • Media queries are a CSS feature.
  • Media queries work by specifying a condition or set of conditions in CSS, if those conditions are met, the associated syeles will be applied.
  • the condition involves the properties of device, such as screen width, height, aspect ratio (aspect ratio of a device refers to relation between width and height. for example a device with aspect raiton of 16:9; means the device width is 19units and height 9 units), device orientation (portrait and landscape orientation), resolution, or even the type of device (e.g., screen, print and speech)
  • media queries consist of media type and media features
  • media type: target type of devices; all: (all devices and media type). screen:(screen with and devices with visual output.) print: (applies when printing documents). speech: (applies to screen reader.)

Syntax

  • @media (condition){

    /* CSS rule applies once the conditions are met */

    }

  • @media (max-width: 600px) { /* css rule for device which has a maximum width of 600 pixels */ body { font-size: 12px; color: red; } }

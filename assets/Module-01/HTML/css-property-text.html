CSS Properties for Text

Other CSS Properties to Style Text

  1. Font properties
    • Font-family: Font face.
    • Font-size: Set the size of text.
    • Font-weight: Determine the thickness of text, such as bold.
    • Font-style: Apply italic or oblique to the text.
  2. Color and background properties
    • Color: Define the color of text.
    • Background-color: Set a background color behind the text.
    • Text-shadow: Add a text shadow effect to the text.
  3. Text formatting properties
    • Text-align: Align text horizontally (left, right, center, justify).
    • Text-decoration: Add visual effects to the text (underline, overline, line-through).
    • Text-transform: Control the capitalization of text (uppercase, lowercase, capitalize).
  4. Spacing and alignment properties
    • Line-height: Adjust the height of each line.
    • Letter-spacing: Change the spacing between characters.
    • Word-spacing: Modify the spacing between words.
    • Word-indent: Set the indentation of the first line of text.
    • Text-justify: Specify the justification method for text.
  5. Text effects and layout properties
    • Text-overflow: Determine how overflowing text is handled.
    • White-space: Control how white space is handled within the text.
    • Word-wrap: Define whether long words can be broken and wrapped onto the next line.
    • Word-break: Specify how to break lines within words.

Colors

  • id: Unique identifier for elements. "id" identifies a single element, while the class attribute can target multiple elements.
    "id" has a higher specificity than the class attribute.

  • style: Used for inline CSS style.

  • href: Defines the destination of a link and the anchor 'a' element.

  • src: Used to link external resources such as images or scripts.

  • alt: Provides text for an image. It is used when the image is not displayed.

  • title: Provides a tooltip when the user hovers over the element.

  • disabled: Makes an input element unresponsive.

  • placeholder: Specifies a short hint in an input text box.

  • required: Makes an input required when filling out forms.

  • value: Sets an initial value for an input element.

  • maxlength: Specifies the maximum number of characters in an input element.

  • target: Specifies where to open a link, either in a new tab or in the same frame.

  • colspan and rowspan: Define the number of columns and rows in an HTML table cell.

  • data: Allows custom data to be added to HTML elements for use in JavaScript.

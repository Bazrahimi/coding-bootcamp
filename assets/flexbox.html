flexbox

flex box, short of Flexible Box Layout Module

  • flexbox is a CSS layount module.
  • provide a flexible way to arrange and align elements within a container.
  • create responsive layouts that can adapt different screen size and orientation.

key concept of Flexbox include:

  1. flex container: the parent element that contain flex items. by applying the display:flex or display:inline-flex
  2. flex Items: the child element within the flex container. this could horizontally or vertically arranged based on flex container's direction.
  3. Main Axis and Cross Axis: the main axis is the primary axis along which flex items are laid out. either horizontally (row direction) or vertically (column direction)
  4. Flex Direction: is define by the main axis. it can be set to row (left to right) or row-reverse (right to left), column (top to bottom) or column-reverse (bottom to top).
  5. Flex Property: control the layoutn and behavour of flex items, including 'justify-conten', 'align-items', 'flex-wrap', 'align-content'.

Event Listeners

Event listeners in JavaScript are functions that are attached to specific events on HTML Event. event listener respond to clicks, mouse movement, key presses, and more.

the process to use event listener.

  1. use method such getElementById, querySelector or other DOM traversal techniques to select HTML elements.
  2. use addEventListener method to attache an event listener to selected element.
  3. event handler or callback functions are the code which will run after when the event listener is completed or proceeed.

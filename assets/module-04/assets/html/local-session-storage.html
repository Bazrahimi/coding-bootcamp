Local and Session Storage

localstorage adn session storage are web storage in the browser. They are part of Web Storage API.

  1. localStorage:
    • store data in browswer even the browser closed or reopened.
    • the Data will remain available in localStorage until it is not clear by the users.
    • localStorage only supported string values.
    • To store data in localStorage, we much use localStorage.setItem(key, value) method.
      to retrieve the values we must use localstorage.getItem(key). we use the same key that we had put for SetItem to retrieve the the getItem.
  2. sessionStorage
    • it is similar to localStorage, but the store data is in session specific.
    • the data will be only available during the session.
    • similar to localStorage, Session storage also store data as key value pair and the value is required to be string only.
    • to stora data we use sessionStorage.setItem(key, Value) method.
      to retrieve the value we need associate the key and use sessionStorage.getItem(key) method.

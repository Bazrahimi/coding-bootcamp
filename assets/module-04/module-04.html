Module-04 Web APIs

APIs or Application programming interfaces which allow web developers to create dynamic, interactive web applications.
APIs are build directly into browsers. when a webpages is loaded in the browser, an object represenation of the webpage (the Document Object Model or DOM) is created. The DOM allows us to use JavaScript to interact with HTML elements.
in this Module, we will learn how to create and manipulate HTML elements, add event listeners, and store data, all using the power of JavaScript.

Flexbox

Flexbox, short for Flexible Box Layout Module

  • Flexbox is a CSS layout module.
  • It provides a flexible way to arrange and align elements within a container.
  • It allows creating responsive layouts that can adapt to different screen sizes and orientations.

Key concepts of Flexbox include:

  1. Flex Container: The parent element that contains flex items. It is defined by applying the display: flex or display: inline-flex property.
  2. Flex Items: The child elements within the flex container. They can be arranged horizontally or vertically based on the flex container's direction.
  3. Main Axis and Cross Axis: The main axis is the primary axis along which flex items are laid out, either horizontally (row direction) or vertically (column direction).
  4. Flex Direction: It is defined by the main axis and can be set to row (left to right), row-reverse (right to left), column (top to bottom), or column-reverse (bottom to top).
  5. Flex Property: It controls the layout and behavior of flex items, including 'justify-content', 'align-items', 'flex-wrap', and 'align-content'.

justify-content: It controls the alignment of flex items along the main axis and accepts the following values:
  • flex-start: Align items to the start of the main axis.
  • flex-end: Align items to the end of the main axis.
  • center: Align items to the center of the main axis.
  • space-between: Distribute flex items evenly along the main axis, with the first item aligned to the start and the last item aligned to the end.
  • space-around: Distribute flex items evenly along the main axis, with equal space around the items.
  • space-evenly: Distribute items evenly along the main axis, with equal space around and between them.

align-items: It controls the alignment of flex items along the cross axis of the flex container. The cross-axis is perpendicular to the main axis. The align-items property accepts the following values:

  • flex-start: Align items to the start of the cross axis.
  • flex-end: Align items to the end of the cross axis.
  • center: Align items to the center of the cross axis.
  • baseline: Align items along their baseline.
  • stretch: Stretch items to fill the container along the cross axis.

In Flexbox, the order property is used to control the visual order of flex items within the flex container.

Transitions

transition in CSS allow an smooth animate changes in property value over a specified duration. it provide a way to addi visual effects and smooth transitions between different state or style of an element.
below some keys aspect of CSS Transitions.

  1. transition Property:
    it specify a CSS property should be animated during a transition.
    hover on me!

  2. duration: specify the length of time that a transition should take in place, in seconds (s) or millisecons (ms).
  3. Timing Fucntion: define the acceleration curve for the transitions. how the visual effect of the transition is going in start, middle and at end of the of the transition
    • the linear timing function have constant animation speed.
    • ease start slow, accelerate in middle, and slow down toward the end.
    • ease-in start slow and accelerate graudually.
    • ease-out start fast and decelerate smoothly.
    • ease-in-out start slow, accerlerate in middle, and decerlerate toward the end.
    • cubic-bezier allows to define a custiom accerleration curve by specifying the control point.
  4. delay:the property set a delay before the transition start.

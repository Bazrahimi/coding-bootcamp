CSS Variables

CSS variables are also known as CSS custom Properties. it is a powerfull feature that allow the developers to use reusable values within css. they function like variables and we can store and reused values throughout the stylesheet. dr CSS variables are declared by using double dash -- prefix followed by a name and assignment value.

The benefits of using CSS variables include.

  1. Reuseablity: you can define a value and reuse it throughout the stylesheet.
  2. Dynamic Changes css varaibles can be modified dynamically (be automatically updated) using JavaScript.
  3. Scope values: it can scoped to specific element. we are able to have different values for diffirent part of douments.

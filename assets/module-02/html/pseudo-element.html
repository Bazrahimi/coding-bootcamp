Pseudo-Elements

Pseudo-Elements are elements that not part of actual HTML markup but can be created and manipulated ujsing css.
Pseudo-Elements are represented by double colons :: in CSS and use with conjunction with CSS Selector to target specific elements.

  1. ::before: insert content before the content of element.
  2. ::after: insert content after the content of element
  3. ::first line: it select and style the first line of text within an element.
    the Pseudo-Elements apply style to the first line
    of paragraph only.
  4. ::first-letter: it select and style the first letter of a text.
  5. ::selection: it allow to style a portion of text.
  6. ::placeholder: element allow us to style the placeholder text inside an input text field.

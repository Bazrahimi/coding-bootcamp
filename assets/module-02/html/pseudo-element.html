Pseudo-Elements

Pseudo-Elements are elements that are not part of the actual HTML markup but can be created and manipulated using CSS.
Pseudo-Elements are represented by double colons (::) in CSS and are used in conjunction with CSS selectors to target specific elements.

  1. ::before: insert content before the content of an element.
  2. ::after: insert content after the content of an element.
  3. ::first-line: selects and styles the first line of text within an element.
    The pseudo-element applies styles to the first line of a paragraph only.
  4. ::first-letter: selects and styles the first letter of text.
  5. ::selection: allows you to style a portion of selected text.
  6. ::placeholder: allows us to style the placeholder text inside an input text field.

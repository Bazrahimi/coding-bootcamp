Typography

Typography in CSS refers to the visual style and arrangement of text on a webpage.


CSS provides several properties to manipulate typography:
  1. font-family: Specifies the font for the text.
  2. font-size: Sets the size of the text.
  3. font-weight: Specifies the thickness or boldness of the text.
  4. font-style: Determines whether the text is italic, oblique, or normal.
  5. color: Sets the color for the text.
  6. text-decoration: Adds decorative effects to the text, such as underline or strikethrough.
  7. text-transform: Modifies the capitalization of the text, such as uppercase or lowercase.
  8. text-align: Aligns the text horizontally within its container.
  9. line-height: Sets the vertical spacing between lines of text.
  10. letter-spacing: Adjusts the spacing between characters.

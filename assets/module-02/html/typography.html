Typography

Typography in CSS refer to visual style and arrangment of the text in webpage.


CSS provide several properties to manipulate typography.
  1. font-family specifify to the font for the text.
  2. font-size set the size of the text.
  3. font-weight: specify thickness or boldness of the text.
  4. font-styl: determine whether the text italic, oblique, or normal.
  5. color: set the color for the text.
  6. tex-decoration: add decorative effects to the text such underline or strikethrough.
  7. text-transform: modifies the capitalization of the text, like uppercase or lowercase.
  8. text-align: align text horizontaly within the container.
  9. line-height: set the vertical spacing between lines.
  10. letter-spacing: adjust the spacing between characters.

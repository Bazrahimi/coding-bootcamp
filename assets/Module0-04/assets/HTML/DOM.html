Document Object Model

DOM is a programming interface that represents the structure of an HTML or XML document as a tree-like structure. It provides a way to access, modify, and manipulate the elements and content of a document dynamically.
When the page is loaded into the browser, the browser creates a representation of the page called the DOM. This allows JavaScript to interact with the document's elements, attributes, and text content.

Window Objects

Window Object is the a global object in web browsers that represent the the current browser window or tab.Using JavaScript we can control and manipulate window object such control navigations, managing cookies and handling events.

here some common properties and methods of the window object

  • window.document: represent the current webpage loaded in the window. we are able to manipulate HTML documents like modifyhing elements, changing style, and handling events.
  • window.innerWidth and window.innerHeight: represent the width and height of the browser window'content area, excluding scrolebars.
    • the widow.innerWidth property returns the width of the browser window content area.
  • window.location: provide information about current URL and alow to Navigate to new URL. it has properties like href(current URL), hostname(domain name), pathname (path of the URL) and methods like assign()(loads new URL) and reload()(reload the current page).
  • window.alert(), window.confrim(), and window.prompt()
    • window.alert(); alert a dialog box.
    • window.confirm(); the window display a confirmation dialog box.
    • window.prompt(): display a prompt dialog box with a message.

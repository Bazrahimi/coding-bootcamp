Variables

In javascript variable are containers which hold value.
it is created by var, let or const.
in JavaScript variable can hold various type of values. some common value that variable can hold them.

  1. numbers: variable can hold numberic values, integers (whole number), floating-point (decimal), infinity, and NaN( Not a Number).
  2. stringvariable can hold stirng. strings are enclosed in single or double quotes.
  3. Booleans: values are true or false.
  4. Arrays; arrays can hold order collection of values. array can hold various values of differnt type inside square bracket [].
  5. Objects: are collection of key values, and object are define inside curly brackets.
  6. function: is block of reuseable code. function can be assigned to a variale called later.
  7. null: is intention empty value.
  8. undefined: varaible that have been declared but not assigned are undefined.

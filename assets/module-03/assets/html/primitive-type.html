Primitive Types

Primitive refer to something basic.
in JavaScript Primitive type is a basic data type that is not have not any other data type. it represent the most basic building block. in contrast, non-primitive type such as object or arrays are having multiple values and have method and properties too.
in Script, there is 6 primitive type. however, there is only one non-primitive type which object. and object has many subtype and variant.

Primitive type in JavaScript

  1. Number: numeric values such as 3, 4.1, Nan(Not a Number)
  2. string: is text enclosed with single and double quote.
  3. Boolian: logical value such as true or false
  4. Nul: intentionally empty value.
  5. Undefined: represent a variable that has been declared but not assigned a value.
  6. symbol: represent unique indentifier. it is used programing to aviod naming conflict.

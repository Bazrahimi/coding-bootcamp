Primitive Types

"Primitive" refers to something basic.
In JavaScript, a primitive type is a basic data type that does not have any other data type. It represents the most basic building block. In contrast, non-primitive types such as objects or arrays have multiple values and also have methods and properties.
In JavaScript, there are 6 primitive types. However, there is only one non-primitive type, which is the object. The object has many subtypes and variants.

Primitive types in JavaScript:

  1. Number: Numeric values such as 3, 4.1, NaN (Not a Number).
  2. String: Text enclosed with single or double quotes.
  3. Boolean: Logical value such as true or false.
  4. Null: Intentionally empty value.
  5. Undefined: Represents a variable that has been declared but not assigned a value.
  6. Symbol: Represents a unique identifier. It is used in programming to avoid naming conflicts.

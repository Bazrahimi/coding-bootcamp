Logical and Comparison Operators

These operators are used to perform logical and comparison operations on values.

Logical Operators:

  1. Logical AND (&&): Returns true (meaning run the code) if both operands are true (meaning if both sides of the condition are met).
  2. Logical OR (||): Returns true (will run the code) if at least one of the operands is true (meaning one side of the condition is met).
  3. Logical NOT (!): Reverses the logical state of an operand (it runs the code if the condition is not true).

    Comparison Operators

  1. Equal (==): Checks if two values are equal. However, it has some unexpected behavior due to type coercion. Type coercion refers to the automatic conversion of one data type to another by the JavaScript engine.
    console.log(5 + "5"); // output: "55".
    Generally, strict equality is preferred.
  2. Strict Equal (===): Checks if two values are equal and does not perform type coercion.
  3. Not Equal (!=): Checks if two values are not equal. Sometimes it does perform type coercion.
  4. Strict Not Equal (!==): Checks if two values or the types of two values are not equal.
  5. Greater than (>): Checks if one value is greater than another value.
  6. Less than (<): Checks if one value is less than another value.
  7. Greater than or equal to (>=): Checks if one value is greater than or equal to another value.
  8. Less than or equal to (<=): Checks if one value is less than or equal to another value.

this keyword

this keyword is a special identifier refers to the context in which the function is executed. we use this keyword to manipulate properies and method of objects that function is calle upon.
this can be any value based context in which the function is called.

  1. Global Scope, this keyword is referenced in global scope outside of any function.

  2. Object Method; when a function is called as a method of an object. this refer to the object itself.

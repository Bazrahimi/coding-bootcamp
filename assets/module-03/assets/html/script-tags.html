Script tags in JavaScript

In JavaScript Script tags are used to include and execute JavaScript code within HTML Documents

  1. Inline Script: you include JavaScript code directly whithin HTML Documents;
    HTML Document: is consist of HTML element and HTML tags, which are used to strucutre and format the content.
    Structure and format of HTML consist of DOCTYPE, HTML element, Head section, Body section, html element and tags and html attribute.
    HTML element: is the ELement plus opening and closing tags of element.
    HTML Tags: are the opening and closing tags, exclude the element itself.
  2. External script: you can reference external JavaScript file using 'src' attributes of the script tag.
  3. Deffered Script by adding the defer attribute to stcript tag, you can indicate that the script should be executed after the HTML document has been parsed.

Conditional Statement

conditional statement allow us to controrl the flow of our

  1. if statement:

    if the condition is met inside the the round bracket and than code will flow inside the curly bracket.
  2. if... else statement:

    if condition is true inside round bracket, the code will run in 1st curly bracket.
    if the condition is not met in round bracket, the code inside the 2nd curly bracket will run.
  3. if...else if...else statement:

    if both condition inside both round bracket is not true. the will run on last curly bracket.
  4. switch statement:

    switch statement handle multiple possible condition.
    these condition satement allow us to control the execution of our code based on specific condition and it make our code more flexible and dynamic.

Conditional Statement

Conditional statements allow us to control the flow of our code.

  1. if statement:

    If the condition inside the round brackets is met, the code inside the curly brackets will be executed.
  2. if...else statement:

    If the condition inside the round brackets is true, the code inside the first set of curly brackets will run.
    If the condition is not met, the code inside the second set of curly brackets will run.
  3. if...else if...else statement:

    If both conditions inside the round brackets are not true, the code inside the last set of curly brackets will run.
  4. switch statement:

    The switch statement handles multiple possible conditions.
    These conditional statements allow us to control the execution of our code based on specific conditions and make our code more flexible and dynamic.
